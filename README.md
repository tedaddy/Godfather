# Godfather

### What it is?
This is a quiz app for Russian teenagers. Model of famous exams, so called "ОГЭ" and "ЕГЭ". I think this program should be helpfull as for teachers so for students.

------------
### Get started

1. Download [Java](https://www.java.com/en/download/ "Java")
2. Download project repository
3. Open folder `product` ( all the rest, if it is not interesting for you, can be deleted )
4. Run `Godfather.jar`
